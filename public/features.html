운영을 가볍게, 관리는 체계적으로.
스튜디오 실무를 돕는 기능 가이드

단순한 예약 시스템을 넘어 스튜디오의 일상적인 반복 업무를 자동화하고,
회원 관리와 매출 데이터를 직관적으로 파악할 수 있는 실무 최적화 환경을 제공합니다.

마케팅 전문 AI, 회계 분석 AI, 경영 전략 AI가 원장님의 데이터를 함께 분석하며,
운영 데이터가 쌓일수록 더 정교한 인사이트와 맞춤형 제안을 드립니다.

📊 1. 스튜디오 통합 관리 대시보드

모든 경영 데이터를 통합하여 직관적인 통계 및 회원 상태를 실시간으로 확인합니다.

📉

매출 현황 및 리포트

  • 매출 추이 시각화: 월별, 일별 매출 등의 결제 데이터를 그래프로 확인하여 흐름을 파악합니다.
  • 수강권별 판매 비중: 권종 및 결제 수단(카드, 현금 등)에 따른 통계를 제공합니다.
  • 재등록 지표 분석: 신규 등록과 만료 예정, 재등록 회원을 분류하여 리텐션(재구매율) 전략에 참고할 수 있습니다.
  • 출석 추이 차트: 요일별·시간대별 출석 패턴을 그래프로 확인하여 수업 편성에 활용합니다.
🏢

다중 지점(Multi-Branch) 운영

  • 지점별 데이터 완벽 분리: 본점과 지점의 시간표, 매출, 강사 데이터를 각각 독립적으로 관리합니다.
  • 통합 확인 지원: 대시보드 내에서 손쉽게 탭을 전환하여 지점별 운영 상황을 체크할 수 있습니다.
  • 지점 추가/삭제: 설정 화면에서 새로운 지점을 자유롭게 추가하거나 정리할 수 있습니다.
  • 각 지점 고유 접속 주소: 스튜디오명.web.app 형태의 전용 도메인이 자동 생성됩니다.
  • 프로그램별 독립 운영: 같은 공간에서도 요가부 / 필라테스부 / 헬스부로 나눠 프로그램별 시간표·매출·강사를 독립 관리할 수 있습니다.
💳

수강권 및 결제 이력 관리

  • 유연한 수강권 설정: 정액권, 횟수권, 요가/필라테스 패키지 등 스튜디오에 맞는 수강권 종류를 설정합니다.
  • 결제 및 사용 시작일 제어: 결제일과 실제 수강 시작일을 다르게 지정할 수 있어 사전 등록이 편리합니다.
  • 수강권 연장 기능: 만료된 수강권도 간편하게 연장 처리할 수 있습니다.
  • 실수 복구(Soft Delete): 실수로 회원이나 결제 기록을 삭제해도 휴지통에서 완전 복구 가능합니다.
  • 수강 횟수 관리 방식 선택: 전체 기간 내 자유 사용 또는 주간 단위(주 N회) 제한 중 선택 가능합니다.
📅

예약 시스템 및 대기열 관리

  • 수업 예약 ON/OFF: 원장님이 설정에서 예약 기능을 켜고 끌 수 있습니다.
  • 정원 관리: 수업당 최대 인원을 지점별/수업별로 세밀하게 설정합니다.
  • 대기열(Waitlist) 자동 승격: 정원이 찬 수업에 대기 신청 → 취소 발생 시 자동 예약 확정 + 알림.
  • 노쇼(No-show) 관리: 예약 후 미출석 시 횟수 차감 규칙을 자유롭게 설정합니다.
  • 예약/취소 마감 시간: 수업 몇 시간 전까지 예약·취소 가능한지 설정할 수 있습니다.
⏸️

회원 홀딩(일시정지) 관리

  • 셀프 홀딩 ON/OFF: 회원이 앱에서 직접 수강권을 일시정지할 수 있도록 허용/제한합니다.
  • 홀딩 규칙 세분화: 수강 기간별 최대 홀딩 횟수, 1회 최대 기간(주 단위)을 자유롭게 설정합니다.
  • 만료일 자동 연장: 홀딩 기간만큼 수강권 만료일이 자동으로 뒤로 밀립니다.
📢

공지사항 및 단체 메시지

  • 공지사항 게시판: 원장님이 작성한 공지가 회원 앱과 강사 앱에 실시간으로 노출됩니다.
  • 단체 메시지 발송: 조건에 맞는 회원들에게 푸시 알림 또는 알림톡을 한 번에 발송합니다.
  • 개별 메시지 이력: 회원별 발송/수신 메시지 이력을 확인할 수 있습니다.
🔄

AI 스마트 데이터 이전 (초기 세팅)

  • 기존 회원 엑셀(CSV) 자동 인식: 이전에 쓰던 회원 목록 파일을 업로드하면 AI가 양식을 자동 분석하여 시스템에 바로 등록합니다.
  • 시간표 이미지 자동 변환: 사진으로 찍은 시간표를 AI가 인식해서 디지털 시간표로 변환합니다.
  • 가격표 이미지 자동 변환: 가격표 사진을 업로드하면 AI가 요금제 데이터로 자동 추출합니다.
⚙️

스튜디오 개성 커스터마이징

  • 로고 업로드: 우리 스튜디오 로고를 업로드하면 키오스크, 회원 앱, 알림 등 모든 곳에 자동 반영됩니다.
  • SNS 링크 관리: 인스타그램, 블로그, 카카오 채널 등 외부 링크를 등록하면 회원 앱에 표시됩니다.
  • 앱 URL 및 QR 코드: 관리자앱/강사앱/회원앱/출석체크앱 각각의 QR 코드를 자동 생성하여 간편 공유를 지원합니다.
  • 운영 가이드 내장: 처음 사용하는 원장님을 위한 단계별 운영 가이드가 앱 내에 탑재되어 있습니다.

📱 2. 회원용 예약 출석 앱

앱스토어 설치 없이 바로 쓸 수 있는 가벼운 전용 웹앱(PWA)으로, 예약부터 정보 확인까지 회원이 직접 처리합니다.

🔑

간편 접속 (PIN 형태 로그인)

  • 직관적인 로그인 방식: 전화번호 기반의 간편 접속을 지원하여 비밀번호 분실로 인한 CS 문의를 줄입니다.
  • PWA(웹앱) 홈화면 설치: 별도 앱스토어를 거치지 않고 브라우저에서 버튼 클릭만으로 스마트폰에 아이콘을 추가할 수 있습니다. iOS/Android 맞춤 안내가 자동 표시됩니다.
📅

투명한 시간표 및 예약 시스템

  • 클래스 잔여석 확인: 수강 신청 가능한 목록과 남은 자리가 실시간으로 표시됩니다.
  • 대기 접수 및 확정: 예약 정원이 가득 차면 대기로 등록하고, 결원 시 순차적으로 푸시 알림과 함께 자동 확정됩니다.
  • 이용 내역 조회: 본인의 수강권 정보(남은 횟수 및 만료일)를 앱 내에서 상시 확인 가능합니다.
  • 가격표 확인: 스튜디오에서 설정한 수강권 가격과 혜택을 앱에서 바로 확인할 수 있습니다.
📊

나의 운동 리포트

  • 출석 통계 시각화: 월별 출석 횟수를 그래프로 확인하여 운동 습관을 점검할 수 있습니다.
  • 출석 이력 조회: 과거 출석 기록을 날짜별로 확인 가능합니다.
  • 수강권 정보 한눈에: 현재 활성 수강권, 남은 횟수, 만료까지 남은 일수를 직관적으로 표시합니다.
  • 결제 이력 조회: 본인의 결제 내역(금액, 결제수단, 날짜)을 투명하게 확인합니다.
⏸️

셀프 홀딩 및 알림 설정

  • 셀프 홀딩(일시정지): 원장님이 허용한 경우, 회원이 앱에서 직접 수강권을 일시정지할 수 있습니다.
  • 푸시 알림 설정: 예약 확정, 수업 리마인더, 공지사항 등의 알림을 개별적으로 켜고 끌 수 있습니다.
  • 공지사항 확인: 원장님이 게시한 공지사항을 앱 내에서 바로 확인합니다.
  • SNS 바로가기: 스튜디오의 인스타그램, 블로그 등 등록된 SNS 링크에 바로 접속할 수 있습니다.

🤖 3. 스마트 출석 데스크 (키오스크)

프론트 인력의 부담을 덜어주는 방문자 확인 및 자동 출석 기록 시스템입니다.

👁️

Edge-AI(기기 내 처리) 안면인식 및 PIN 병행 출석

  • 안면인식 모드 (Face-API 기반): 얼굴 사진을 서버로 보내지 않고 태블릿 자체에서 인식 처리합니다(Edge Computing). 비접촉 방식으로 빠르게 출석을 기록합니다.
  • PIN(전화번호 뒷자리) 입력 병행: 마스크, 조명 등으로 인식이 어려울 때 숫자 패드로 즉시 우회 가능합니다.
  • 중복 체크인 방지: 짧은 시간(5초) 내 중복 인식을 자동 제어하여 출석 카운트가 여러 번 차감되지 않습니다.
  • TTS(음성 합성) 안내: 정상 출석/만료/중복 출석 시 각각 다른 음성으로 자동 안내합니다.
🔒

안면정보 암호화 (개인정보보호법 준수)

  • 128차원 벡터(숫자 배열) 변환 저장: 원본 사진을 파일로 저장하지 않고 역추적 불가능한 숫자 배열 형태만 보관합니다.
  • 회원 정보 실시간 매칭: 출석과 동시에 수강권 상태(만료/잔여 횟수)를 확인하여, 소진 시 입장 제한 알림을 화면에 표시합니다.
  • 오프라인 출석 로컬 캐싱: 인터넷이 끊겨도 LocalStorage에 회원 데이터가 캐싱되어 출석이 계속 기록됩니다. 연결 복구 시 자동 동기화.
🖼️

키오스크 공지 및 미디어 갤러리

  • 이미지/영상 공지: 태블릿 대기 화면에 이미지(최대 10MB)나 영상(MP4/WebM/MOV, 최대 50MB)을 표시할 수 있습니다.
  • 미디어 갤러리 관리: 여러 미디어를 업로드해두고 갤러리에서 선택 → 즉시 키오스크에 반영합니다.
  • 터치로 출석부 전환: 공지 화면을 터치하면 자동으로 출석체크 화면으로 전환됩니다.
⏰

키오스크 세부 제어

  • 운영 시간 자동 ON/OFF: 설정된 시간 외에는 화면과 카메라를 끄고 절전(대기) 모드에 진입하여 태블릿 수명을 극대화합니다.
  • 근접 감지 자동 복귀: 카메라로 사람 접근을 감지하면 자동으로 공지 화면에서 출석체크 화면으로 전환합니다.
  • 키오스크 로고 커스텀: 숫자패드 위에 표시될 로고를 최대 2개까지 등록하고, 배경색(흰색/검정/투명)과 농도를 세밀하게 조절합니다.
  • 카메라 프리뷰 크기 설정: 카메라 영상을 크게(로고 하단) 또는 작게(옆에 미니 프리뷰) 표시할 수 있습니다.

🧘‍♀️ 4. 강사 전용 출석 및 수업 화면

운영 매출 등의 민감 정보를 제한하고, 맡은 수업의 관리 권한만 부여합니다.

✅

출결 상황 연동

  • 클래스별 명단 확인: 오늘 로그인한 강사 담당의 수업과 수강 신청자 명단만 깔끔하게 노출됩니다.
  • 수동 출석 및 지각 체크: 키오스크 이용이 어려웠던 회원을 강사가 앱 화면에서 직접 출석 처리할 수 있습니다.
  • 특이사항(메모) 공유: 부상 여부 등 회원별 특이사항을 적어두어 수업 퀄리티 유지와 관리에 참고할 수 있습니다.
  • 공지사항 확인: 원장님이 게시한 공지사항을 강사 전용 앱에서도 확인할 수 있습니다.
🔐

권한 분리 및 프라이버시 보호

  • PIN 기반 인증(커스텀 토큰): 강사님들은 전화번호 뒷자리(PIN)로 간편하게 로그인합니다. 일반 회원과는 다른 고유 인증 체계입니다.
  • 매출 정보 비공개: 스튜디오 전체 매출이나 다른 강사의 수업 정보는 일체 볼 수 없습니다.
  • 담당 수업만 노출: 본인이 맡은 수업 명단과 수동 출석 체크 권한만 가집니다.

🧠 5. AI 비서 기능 (대화형 어시스턴트)

엑셀을 열 필요 없이 채팅하듯 물어보면 AI가 즉시 답해드립니다.

💬

자연어 대화형 AI 비서

  • 대화로 경영 데이터 조회: "오늘 밤 수업에 10회권 이상 남은 사람은?", "김복순 회원님 잔여 횟수는?" 채팅하듯 물어보면 AI가 데이터를 검색해 즉시 대답합니다.
  • 매출 요약 리포트: 오늘, 이번 주, 이번 달 매출을 AI가 자동으로 요약하여 한눈에 파악할 수 있습니다.
📈

이탈 방지 및 리텐션 분석

  • 이탈 위험 회원 자동 감지: 만료 예정이거나 최근 14일간 나타나지 않은 회원을 AI가 자동으로 추려냅니다.
  • 안부 문자 초안 작성: 부담스럽지 않은 톤의 안부 메시지 템플릿을 AI가 자동 제안하여 원장님의 소통 부담을 덜어드립니다.
  • 재등록률 실시간 시각화: 재등록률과 이탈률을 그래프로 확인할 수 있습니다.

☁️ 6. 알림 시스템 및 데이터 보안

안정적인 서버 환경과 회원과의 투명한 소통을 위한 도구입니다.

💬

통합 메시징 플랫폼

  • 앱 푸시(Push) 무료 제공: 안내 사항, 휴강 안내, 예약 확정/취소 등을 무료 앱 기반 FCM(Firebase Cloud Messaging) 푸시로 우선 발송합니다.
  • 알림톡 / SMS 보완: 비즈뿌리오 API 연동 및 고정 IP(Cloud NAT)를 통해 스팸 필터링을 최소화하고 높은 도달률을 확보합니다.
  • 발송 이력 관리: 푸시/SMS/알림톡 발송 이력을 대시보드에서 확인할 수 있습니다.
  • 카카오톡 상담 버튼: 회원 앱 하단에 카카오톡 상담 채널 바로가기 버튼이 내장되어 있습니다.
🛡️

데이터 안전 관리 및 인프라

  • Google Firebase 클라우드: 구글 클라우드(asia-northeast3, 서울 리전) 인프라를 활용해 데이터 동기화와 저장소 확장을 유연하게 처리합니다.
  • Firestore 자동 백업 (하루 2회): 운영 데이터가 매일 2번 자동 백업되어, 만약의 사태에도 데이터를 복구할 수 있습니다.
  • 🔒 외부 공격 방어: Google Cloud의 엔터프라이즈급 보안 인프라(DDoS 방어, SSL/TLS 암호화, Firestore 보안 규칙)가 스튜디오 데이터를 24시간 보호합니다.
  • 🛡️ 실수로부터의 보호: 회원 정보를 삭제해도 즉시 복구(Soft Delete)가 가능하며, 매출 데이터는 변경 이력이 자동 기록되어 되돌릴 수 있습니다.
  • 🌐 천재지변·장애 대비: 서울 리전 장애 시에도 Google Cloud의 다중 영역(Multi-Zone) 복제를 통해 데이터가 안전하게 보존됩니다. 백업 데이터로 빠른 복구가 가능합니다.
  • 오프라인 출석 자동 동기화: 와이파이가 끊겨도 출석은 LocalStorage에 기록되고, 연결 복구 시 자동 동기화됩니다.
  • 네트워크 상태 모니터링: 인터넷 연결이 끊기면 앱 상단에 알림을 표시하고, 복구 시 자동으로 데이터를 동기화합니다.
📱

PWA(웹앱) 4개 독립 앱 제공

  • 앱스토어 없이 즉시 설치: 사파리나 크롬에서 "홈 화면에 추가"만 누르면 출석체크/회원용/강사용/원장님용 4개의 독립 아이콘이 설치됩니다.
  • 스튜디오 로고 자동 반영: PWA 아이콘과 스플래시 화면에 우리 스튜디오 로고와 이름이 자동 적용됩니다. (SSR 동적 매니페스트 생성)
  • SNS 공유 최적화(OG 태그): 카카오/라인/페이스북 등에 앱 링크를 공유하면 스튜디오 로고와 이름이 미리보기에 예쁘게 표시됩니다.
  • 다국어 지원: 한국어/영어 UI 전환을 지원하며, 외국인 회원도 편하게 이용할 수 있습니다.
⚡

성능 최적화 및 안정성

  • 레거시(구형) 브라우저 지원: 오래된 태블릿에서도 안정적으로 동작하도록 Polyfill(호환성 보완 코드) 및 Legacy 빌드를 자동 적용합니다.
  • 에러 자동 복구: 앱에서 오류 발생 시 자동으로 복구를 시도하고, 화이트 스크린(빈 화면)을 방지합니다.
  • Warmup(사전 로딩): 앱 시작 시 필수 데이터를 미리 로드하여 체감 속도를 높입니다.
  • SaaS 멀티 테넌트 아키텍처: 하나의 서버에서 여러 스튜디오가 완벽하게 격리된 상태로 운영됩니다. 데이터가 섞일 걱정이 없습니다.
  • 개인정보처리방침: 법적 요구사항에 맞는 개인정보처리방침 페이지가 자동 제공됩니다.
안내: 이 페이지에 설명된 서비스 구성 및 부가 기능(얼굴 인식, 메시지 자동화 등)은 실제 데모 페이지에서 대부분 동일하게 체험 가능하도록 구현되어 있습니다. 시스템 한도 및 통신사 환경에 따라 세부적인 통지 속도는 일부 편차가 있을 수 있습니다.
