무료 데모 시작하기

오직 스튜디오 원장님들을 위해 설계했습니다.

"강사님들 권한은 어떻게 분리하지? 2호점 매출은 어떻게 합산해서 볼까? 노쇼(No-show)하는 회원은 어떻게 관리해야 상처받지 않고 안내할 수 있을까? 비싼 지문 인식기를 사야만 무인 출석이 가능할까?"

원장님들이 현장에서 겪는 스트레스와 페인포인트(Pain-point)를 누구보다 잘 알기에, **개발자가 아닌 요가·필라테스 전문가의 시선**에서 이 모든 고민을 단 하나의 시스템에 담아냈습니다.

🤖 초고속 Edge-AI 무인 출석 (키오스크)

고가의 장비나 복잡한 지문 등록이 필요 없습니다. 태블릿 한 대로 운영되는 마법 같은 AI 출석.

⚡
0.1초 안면 인식 & PIN 병행
카메라 앞에 서기만 하면 출석 끝. 마스크나 조명 변수로 인식이 어려울 땐 휴대폰 뒷자리(PIN)로 즉시 우회 가능합니다.
  • Face-API 기반 오프라인 Edge 추론
  • 정상/만료/중복 출석 자동 음성 반환(TTS)
  • 절전 모드 자동화로 태블릿 수명 극대화
📶
강력한 오프라인 복원력
와이파이가 끊겼나요? 당황하지 마세요. 오프라인에서도 작동하고, 화면에서 통신 장비를 재부팅하도록 스스로 대처합니다.
  • 오프라인 출석 캐싱 (연결 시 즉시 서버 동기화)
  • 출석 데이터 유실 0% 아키텍처

🧠 내 손안의 비서, AI 어드민

수십 개의 버튼을 누르고 엑셀을 열 필요가 없습니다. 묻고, 대답을 듣고, 결정을 내리세요.

💬
자연어 대화형 AI 비서
"오늘 밤 수업에 10회권 이상 남은 사람 누구야?", "김복순 회원님 잔여 횟수는?" 채팅하듯 물어보면 AI가 데이터베이스를 검색해 즉시 대답합니다.
  • Gemini / OpenAI LLM 기반 구문 분석
  • 매출 요약 및 특정 회원 정보 즉각 조회
📈
이탈 방지 및 리텐션 분석
만료 예정된 회원, 최근 14일간 나타나지 않은 회원을 AI가 자동으로 추려내어, 부담스럽지 않은 안부 문자 초안까지 작성해 줍니다.
  • 재등록률 / 이탈률 실시간 시각화
  • 휴지통(Soft Delete) 기능으로 안전한 데이터 관리

🏢 다중 지점(Multi-Branch) & PWA

본점과 수많은 지점의 벽을 허물었습니다. 엔터프라이즈의 스케일을 소규모 스튜디오에.

🌐
지점별 완벽 독립 시스템
로그아웃/로그인 없이 상단 탭만 누르면 홍대점, 강남점의 매출과 스케줄을 넘나듭니다. 하지만 강사와 회원은 자신의 지점만 보게 됩니다.
  • 프랜차이즈급 테넌트 분리 아키텍처
  • 각 지점별 커스텀 도메인 (xxx-yoga.web.app) 제공
📱
권한별 3개의 독립 PWA 앱
앱스토어 승인 대기 없이, 사파리나 크롬에서 버튼 한 번이면 바탕화면에 회원용/강사용/관리자용 독립 앱이 설치됩니다.
  • 설치율이 높은 가벼운 웹앱 (Progressive Web App)
  • 스튜디오 로고와 테마 컬러가 앱 아이콘에 자동 반영

🛡️ 극도의 단순함, 철벽같은 정보 보호

개발자가 영혼을 갈아넣어 만든 가장 복잡한 클라우드 기술이, 사용자에겐 가장 단순한 클릭 한 번으로 느껴집니다.

개인정보를 저장하지 않는 생체인식

회원의 미소 띤 얼굴 사진은 서버에 저장되지 않습니다. PassFlow Ai는 사진을 128개의 무의미한 숫자 배열(Vector)로 즉시 암호화 변환하며, 어떠한 경우에도 원본 얼굴을 복원할 수 없습니다. (개인정보보호법 완벽 준수)

강사 프라이버시 보호

강사님들은 전용 앱을 통해 로그인합니다. 스튜디오의 민감한 전체 매출이나 다른 강사의 수업 정보는 볼 수 없으며, 오로지 본인의 수업 명단과 수동 출석 체크 권한만 가집니다.
