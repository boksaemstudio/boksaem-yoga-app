완벽한 스튜디오 통제망

단순한 관리 프로그램을 넘어, 관리자·강사·회원을 연결하는 4개의 앱 생태계를 구축합니다.
이탈 예측부터 휴지통 복원 시스템까지, 운영에 필요한 모든 세부 스펙을 공개합니다.

👨‍💼 1. 마스터 관리자 대시보드 (Admin Dashboard)

경영자가 스튜디오의 모든 흐름을 원격으로 장악하고, 반복 업무를 AI에 위임하는 관제탑입니다.

🧠

1.1 AI 지능형 경영 브리핑

  • 실시간 상황 인지: 오전/오후 시간에 따라 문맥을 변경하며 동적 메시지 생성
  • 휴면 회원 추적 (이탈 예측): 최근 14일 이상 미출석한 회원의 이탈 위험도 자동 산출 및 리스트업
  • AI 스마트 제안 엔진: 장기 미접속 회원 리스트를 분석, 원클릭으로 보낼 수 있는 '안부 메시지' 초안을 자동 승인 대기열에 상신
  • 데이터 추이 보고: 전월/현재의 일일 매출 속도를 비교해 목표 초과 달성 여부 예측 분석
📊

1.2 운영 공간(Multi-Space) 분리 관리

  • 공간별 격리 프레임워크: '본점', '2호점' 등 물리적 지점은 물론, 같은 건물 내 '필라테스 기구실', '매트 요가실', '플라잉 실크 룸' 등 프로그램 성격이 다른 공간을 각각 독립된 운영 단위로 등록하여 개별 키오스크 배치 가능
  • 공간별 독립 시간표: 각 운영 공간마다 강사 배정, 수업 시간, 정원, 예약 정책을 별도로 캡슐화하여 관리 혼선 완전 방지
  • 공간 스위칭 관제탑: 하나의 관리자 대시보드에서 모든 운영 공간의 실시간 현황을 원터치로 전환하며 모니터링
  • 통합 매출 집계 + 공간별 성과 교차 비교: 전체 합산 매출 통계와 개별 공간별 성장세를 교차 비교하는 데이터 시각화 패널 제공
💳

1.3 빈틈없는 수강권 & 결제 제어

  • 다이내믹 수강권 설정: 월 구독형, 패키지(횟수), 무제한 이용권, 현금/카드 교차 할인가 개별 셋업
  • 위약금 및 환불 자동 계산: 남은 횟수/기간에 비례해 약관에 맞춘 양도비 및 중도 해지 위약금 수식 내장
  • 마이너스(-) 크레딧 허용 제어: 정책에 따라 수강권이 약간 오버되어도 임시 예약을 허용/차단 가능
  • 히스토리 무결성: 회원의 모든 수강 내역(결제, 차감, 출석) 데이터 타임라인 방식 영구 추적 보존
🛡️

1.4 휴지통(Trash) 복구 및 보안 인프라

  • 소프트 딜리트 (Soft Delete): 실수로 회원이나 결제 내역을 삭제해도 완전 파기되지 않고 '휴지통' 탭으로 분리
  • 1초 롤백 복원: 휴지통에서 언제든 삭제 당시 상태 그대로 100% 원복 가능 (결제액, 횟수 유지)
  • 자동 데이터 스냅샷: Cloud Functions 연동을 통해 매일 새벽 3시 DB 백업 상태 무결성 검증 통과 완료
  • 기기별 강력한 권한 제어: 관리자만 접근 가능한 특수 Dashboard URL 방화벽 보호

📱 2. 회원 전용 모바일 룸 (Member App)

상담원이 필요 없는 자율 예약/관리 시스템입니다. PWA 기술로 앱스토어 심사 없이 1초만에 폰에 설치됩니다.

⚡

2.1 초고속 마찰 제로 예약망

  • PIN 로그인 (Zero-Friction): 아이디/비밀번호 찾기 장애를 없앱니다. 본인 폰번호 뒷자리(4자리)로 즉시 모바일 접근 허용
  • 실시간 클래스 정원 제어: 수강 잔여석이 0석일 시 붉은색 표시와 함께 자동으로 예약 버튼을 [대기접수] 모드로 강제 스위칭
  • 복합 수강권 지원: 필라테스 기구권 10회, 요가 무제한 30일 등 여러 수강권을 탭(Tab)별로 확인 및 각각의 정책에 맞춰 예약 차감
🔁

2.2 대기 예약 오토-릴레이 엔진

  • 우선순위 대기열 (Queue): 만석인 수업에 대기 등록 시 실시간 대기 순번 부여 (내 순번 즉시 확인 가능)
  • 취소 즉시 자동 확정: 기존 예약자가 취소하는 순간, 0.1초 랙 없이 다음 순번을 자동 [예약 확정] 상태로 리프팅
  • 푸시 및 알림톡 발송: 대기가 확정되면 해당 회원 스마트폰으로 PWA 백그라운드 푸시와 카카오 알림톡 동시 타격
📸

2.3 얼굴 인식(Face ID) 출석 패스

  • 셀프 생체 정보 등록: 회원 앱 내에서 스마트폰 카메라로 본인의 정면 얼굴 특징점(Vector Descriptor) 칩핑 등록 가능
  • 최적 프레임 추출: 서버로 얼굴 사진을 넘기지 않고 브라우저 단에서 수학적 벡터 데이타(128차원)만 생성하여 개인정보 침해 요소 제거
  • AI 스캐닝 입장 지시: 스튜디오 입구 키오스크 화면을 쳐다보기만 해도 0.2초 내로 매칭되어 센터 자동 출석 완료 처리

🧘‍♀️ 3. 강사용 하이브리드 교탁 (Instructor App)

수업 진행에만 집중하십시오. 원장님의 매출 데이터를 숨기고 강사 고유의 클래스 명단만 노출합니다.

📋

3.1 배정된 클래스 타겟팅 뷰

  • 개인화 된 스케줄 파싱: 전체 스튜디오 스케줄러 중 로그인한 강사 이름과 매칭된 시간표와 수강생만 필터링하여 노출 (정보 오입력·유출 차단)
  • 수업별 예약자 프로필 조회: 회원의 이름, 성별, 수강권 만료 유무, 특이사항(부상/주의)을 버튼 클릭 한 번에 브리핑
  • 실시간 컬러 마커: 출석 대기(회색), 출석 완료(녹색), 결석(적색) 등 현재 회원의 입장 상태 실시간 하이라이트 동기화
✒️

3.2 강권 출석 제어 및 시퀀스 노트

  • 비대면 키오스크 대체(강제 마킹): 안내데스크 미배치 샵이나 기계 고장 시, 강사가 스마트폰에서 터치 한 번으로 체크인 통과 명령 전송
  • 클래스 히스토리 노트: 당일 수업 피드백, 시퀀스 루틴, 회원의 난이도 체감 불만 사항 등을 노트에 기록 가능 (이 기록은 관리자 대시보드와 자동 연동)
  • 개별 수강생 평가록 (VIP 관리): 문제성 회원이나 장기 미참석 조짐이 보이는 회원에게 주의/알람 플래그 부여 가능

🤖 4. 입구 무인 안면인식 센터 (Smart Kiosk)

리셉션 직원의 인건비를 0으로 만듭니다. 태블릿이나 웹캠 연결 PC 한 대면 완벽하게 가동됩니다.

👁️

4.1 듀얼 인증 보안망 (얼굴 & 핀셋)

  • 페이스 캡처 (Face-API): 웹캠 근처 1미터 이내로 사용자가 들어오면 고속 안면 트래커가 작동하여 명단 속 수백 명 중 일치자 색출 (정확도 99% 텐서 인퍼런스)
  • 투스텝 PIN 백업: 안경 교체, 마스크, 극심한 조명 변화 등으로 생체 인식 우회 시 연락처 4자리 눌러 즉시 통과 지원
🗣️

4.2 AI 개인화 음성 그리팅 엔진 (TTS)

  • 맞춤형 음성 피드백: "복순 회원님 환영합니다." 단방향 알림을 넘어, "복순님, 일주일 만이네요!" 형태의 날짜 차이 모듈 연산에 따른 AI 음성 안내 조합
  • 예외 차단 시스템 (Gatekeeper): 예약 시간이 아니거나, 수강 잔여 횟수가 0회인 회원이 출석 시, 스크린에 경고(적색 UI) 및 접근 불가 사유를 출력하여 미승인 출입 원천 방지
