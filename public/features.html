앱별 데모체험 핵심 기능 도입 문의 홈으로

전체 스펙 명세서

PassFlow는 4가지 독자적인 앱 환경을 모두 제공합니다.
각 역할에 최적화된 상세하고 강력한 기술 기능을 확인해 보세요.

👨‍💼 1. 통합 관리자 대시보드 운영 (Admin App)

스튜디오의 모든 흐름을 장악하는 관제탑. 반복 업무를 AI와 자동화 로직에 위임합니다.

📊

실시간 운영 대시보드

  • 당월 예상 총 매출, 실제 입금액, 미수금 현황 자동 집계
  • 기간별(주/월/년) 매출 추이 및 활성 회원 수 그래프 시각화
  • 일일 클래스 점유율 및 출석률 실시간 모니터링
  • AI 기반 스튜디오 일일 경영 브리핑 리포트 매일 자동 생성
👥

빈틈없는 회원 관리 (CRM)

  • 수강권별(횟수제/기간제) 자동 차감 및 만료일 직관적 표시
  • 가족 쉐어링 지원: 한 수강권을 여러 명이 분할 사용 가능
  • 결제 히스토리, 이전 수강권 내역 평생 추적 (Soft Delete 적용)
  • 상담 기록 및 특이사항 등 비밀 메모 기능
📅

탁월한 스케줄링 시스템

  • 강남점/홍대점 등 여러 지점 동시 멀티 테넌트(Multi-Tenant) 탭 관리
  • 반복적인 시간표를 위한 '지난달 스케줄 이월 복사' 기능
  • 대기 예약자 자동 승급 및 알림발송 처리 로직 내장
  • 특정 클래스에 강사 단독 배정 및 권한 제어 부여
⚠️

스마트 정산 및 데이터 보호

  • 잔여 수강권 일수/횟수 대비 양도·중도 환불 위약금 자동 수식화 계산
  • 관리자의 실수로 지워진 회원을 되살리는 휴지통 복원(Trash/Rollback) 시스템
  • 24시간 주기 자동 DB 스냅샷 백업 수행 (Cloud Functions 연동)
  • CSV 파일 대량 익스포트 및 오프라인 명부 출력 (엑셀 지원)

🧘‍♀️ 2. 강사 전용 모바일 룸 (Instructor App)

수업 준비와 수강생 집중을 위한 가벼운 교탁. 오직 나에게 할당된 수업만 보입니다.

📱

앱스토어가 필요 없는 PWA 전용앱

  • 설치 과정 없이 QR코드나 링크 하나로 스마트폰 바탕화면에 즉시 앱 설치(PWA) 지원
  • 원장님의 코어 관리자 권한과 물리적으로 분리된 안전한 샌드박스 환경 제공
👀

할당된 수업 스케줄 다이렉트 뷰

  • 본인이 배정된 당일 또는 주간 클래스만 깔끔하게 목록화 되어 노출
  • 현재 예약 확정된 수강생의 명단(이름/잔여횟수 등)을 미리 확인
✅

비대면 하이브리드 수동 출석

  • 안내데스크 키오스크가 고장나거나 혼잡할 때, 강사가 휴대폰에서 버튼 터치로 직접 출석/결석 강제 처리 가능
  • 출석 여부에 따른 컬러 하이라이팅으로 즉각적 시각 안내
📝

프라이빗 클래스 리뷰 및 시퀀스 노트

  • 당일 진행한 수업의 시퀀스나 클래스 전반적인 난이도를 비공개 메모
  • 특정 회원에 대한 부상 부위나 특이점 기록 연동 (관리자와 공유됨)

📱 3. 스마트 회원 뷰 (Member Web App)

문자/카톡 문의 없이도 회원들이 스스로 통제하고 예약하는 자율 시스템입니다.

📆

1초 컷 직관적인 예약 시스템

  • 별도 로그인 절차 없이, 원장님이 부여한 보안 PIN 번호로 즉시 로그인
  • 날짜별 클래스 정원/잔여석 실시간 연동 (만석 시 붉은색 표시 등)
  • 클릭 2번만으로 이루어지는 당일/주간/월간 클래스 다이렉트 예약 확정
⏳

대기자 명단 스마트 자동 릴레이

  • 클래스 정원 만석 시 시스템 상 대기자로 등록 가능
  • 기존 예약자가 수강 취소 시, 시스템이 대기 순번에 따라 자동 예약 확정 후 카톡/푸시 알림 실시간 발송
💳

투명한 마이페이지 (수강권 & 내역 조회)

  • 사용 중인 다수의 수강권(요가 10회권, 필라테스 기구 3달권 등) 각각의 잔여 상태 개별 조회 가능
  • 과거 6개월 치 본인의 출석 히스토리를 캘린더 피드로 시각화
🔔

만료 전 넛지 알림 (Retention 기능)

  • 수강권 기한 만료 임박, 또는 잔여 횟수가 2회 이하로 떨어지면 자동으로 회원에게 재연장 안내 알림 팝업 전송

🖥️ 4. 스튜디오 프론트 태블릿 모니터 (Kiosk App)

안내데스크에 아이패드나 태블릿 단 한 대만 켜두면, 무인 출석이 완성됩니다.

🔢

하이브리드 다중 인증 모드

  • 빠르고 직관적인 PIN 번호(ex: 뒷자리 4개) 방식 지원
  • 카메라 기반 AI 오프라인 얼굴인식 보정 출석 지원 (선택 옵션)
  • 오차 없는 동명이인 처리 로직 (번호 뒷자리 충돌 시 안전 UI 지원)
🔊

상황별 TTS 음성 피드백 연동

  • 단순 팝업을 넘어 "김XX 회원님 출석되었습니다" 와 같은 스마트 음성 즉시 송출 기능 적용
  • "수강권이 만료되었습니다", "수강권 등록이 필요합니다" 등 엣지케이스별 에러 대응 음성 지원
🌐

끊김 방지 오프라인 데이터 생존 모드

  • 스튜디오 와이파이(Wi-Fi)가 불안정하거나 인터넷이 일시적으로 끊어져도 로컬 스토리지 엔진이 출석 데이터를 영구 캐싱함
  • 인터넷 복구 순간 메인 서버(Firestore)로 수십 건의 대기 패킷을 1초 만에 일괄 동기화 및 누락 방지 처리
🤖

다이나믹 AI 인사말 위젯

  • 오늘 날씨 정보와 계절, 시간대(아침반/저녁반)를 AI가 분석하여, 출석 후 화면에 자동 인사말과 컨디션 추천 팝업 로딩 (캐싱을 통해 속도 지연 0초 실현)
