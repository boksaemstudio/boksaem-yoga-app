# PassFlow 다국어 번역 요청

> 아래 영어 텍스트를 각 언어로 번역해주세요.
> 요가/피트니스 스튜디오 관리 SaaS 플랫폼에서 사용되는 UI 텍스트입니다.
> `{변수명}` 형태의 템플릿 변수는 절대 번역하지 말고 그대로 유지하세요.
> 결과는 아래와 동일한 JSON 형식으로 출력해주세요.

## 출력 형식
```json
{
  "언어코드": {
    "키": "번역된 텍스트"
  }
}
```
